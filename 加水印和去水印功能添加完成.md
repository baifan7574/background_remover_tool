## 🧪 测试步骤

1. **启动服务器**:
   ```bash
   # 后端
   python sk_app.py
   
   # 前端
   cd frontend
   python -m http.server 8000
   ```

2. **测试加水印功能**:
   - 登录账户
   - 打开"加水印"工具
   - 上传图片
   - 输入水印文字（如："© 2025"）
   - 选择位置（如：右下角）
   - 调节透明度（如：50%）
   - 选择字体大小（如：40）
   - 选择字体颜色（如：白色）
   - 点击"开始处理"
   - 查看结果并下载

3. **测试去水印功能**:
   - 登录账户
   - 打开"去水印"工具
   - 上传带水印的图片
   - 点击"开始处理"
   - 查看结果并下载

## 📝 注意事项
