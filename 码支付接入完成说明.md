## ⚠️ 本地开发注意事项

### **问题：本地无法接收回调通知**

**原因：**
- 本地地址 `http://localhost:5000` 无法被码支付平台访问
- 回调通知无法送达

**解决方案：**

#### **方案1：使用内网穿透（推荐测试）**
1. 下载 ngrok: https://ngrok.com/download
2. 启动 ngrok:
   ```bash
   ngrok http 5000
   ```
3. 复制生成的公网地址，例如：`https://abc123.ngrok.io`
4. 修改 `sk_app.py` 中的通知地址：
   ```python
   notify_url = f"https://abc123.ngrok.io/api/payment/notify"
   ```

## 🔐 签名算法说明

根据您提供的API文档，签名算法如下：

1. **参数排序**：将所有参数（除`sign`和`sign_type`）按参数名ASCII码从小到大排序
2. **拼接字符串**：`key1=value1&key2=value2`
3. **添加密钥**：`key1=value1&key2=value2&key=商户密钥`
4. **MD5加密**：对拼接后的字符串进行MD5加密，转小写

**示例：**
```python
参数：
pid=10294
type=alipay
out_trade_no=ORDER123
money=19.00
name=入门版-月付

排序后：
money=19.00&name=入门版-月付&out_trade_no=ORDER123&pid=10294&type=alipay

添加密钥：
money=19.00&name=入门版-月付&out_trade_no=ORDER123&pid=10294&type=alipay&key=X0cJyf2G0EjDKtQe9NMf

MD5加密（小写）：
sign = md5(拼接字符串).lower()
```
