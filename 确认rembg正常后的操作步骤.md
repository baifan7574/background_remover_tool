## 🔍 如果后端启动时显示rembg未安装

如果后端启动时显示 `⚠️ rembg库未安装`，但你的测试脚本显示rembg正常，可能是：

**问题**：后端服务器使用的Python环境和你测试时使用的Python环境不同

**解决方法**：
1. 检查后端服务器启动时使用的Python
   - 查看后端启动日志的第一行，应该显示Python版本
   - 或者查看 `启动后端服务器.bat` 中使用的python命令

2. 在后端服务器使用的Python环境中安装rembg：
   ```bash
   # 如果后端使用python，运行：
   python -m pip install rembg
   
   # 如果后端使用python3，运行：
   python3 -m pip install rembg
   ```

3. 或者，在测试脚本使用的Python环境中启动后端服务器
