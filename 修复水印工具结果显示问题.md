# ✅ 修复水印工具结果显示问题

## 🔧 修复的问题

### 问题描述
用户报告：加水印和去水印工具处理完成后没有对话框跳出来，无法下载结果。

### 问题原因
1. **API URL错误**: `processImage` 函数中的 switch 语句缺少 `add_watermark` 和 `remove_watermark` 的 case，导致请求被发送到错误的 API (`background-remover`)
2. **结果显示缺失**: `showSuccessResult` 函数中的 switch 语句缺少这两个工具的处理，导致结果无法正确显示

### 修复内容

#### 1. 修复 `processImage` 函数 ✅
**文件**: `frontend/js/app.js`
**位置**: 第1134-1151行

**添加了**:
```javascript
case 'add_watermark':
    apiUrl = `${this.apiBaseUrl}/api/tools/add-watermark`;
    this.updateProgress(20, '准备水印参数...');
    break;
case 'remove_watermark':
    apiUrl = `${this.apiBaseUrl}/api/tools/remove-watermark`;
    this.updateProgress(20, '准备去水印参数...');
    break;
```

#### 2. 修复 `showSuccessResult` 函数 ✅
**文件**: `frontend/js/app.js`
**位置**: 第2231-2385行

**添加了**:
- `add_watermark` case: 显示加水印结果，包含图片预览和下载按钮
- `remove_watermark` case: 显示去水印结果，包含图片预览和下载按钮
