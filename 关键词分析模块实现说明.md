## 🔑 需要申请的API密钥

### 1. **OpenAI API密钥**（必需 - 用于GPT关键词提取）

**用途**：
- 提取跨平台关键词
- 生成关键词建议
- 长尾关键词挖掘

**如何申请**：
1. 访问：https://platform.openai.com/
2. 注册账号
3. 进入 API Keys 页面
4. 创建新的密钥
5. 复制密钥（格式：`sk-...`）

**费用**：
- GPT-3.5-turbo：约 $0.002/1000 tokens（非常便宜）
- 每次关键词提取约消耗 500-1000 tokens
- 成本：约 $0.001-0.002 每次

**是否需要**：
- ✅ **必需** - 没有API密钥无法使用GPT功能

---

### 2. **亚马逊SP-API密钥**（必需 - 用于竞争度查询）

**用途**：
- 查询关键词竞争度
- 获取搜索量数据
- 竞品关键词对比

**如何申请**：
1. 访问：https://developer.amazon.com/sp-api/
2. 注册开发者账号
3. 创建应用（Application）
4. 获取以下信息：
   - `LWA Client ID`
   - `LWA Client Secret`
   - `Refresh Token`
   - `AWS Access Key ID`
   - `AWS Secret Access Key`
   - `Marketplace ID`（如：US = A1PA6795UKMFR9）

**费用**：
- 免费配额：每月有限额度
- 超出后按请求收费（价格较低）
- 通常免费额度足够使用

**是否需要**：
- ✅ **必需** - 没有API密钥无法获取真实的亚马逊数据

**申请难度**：
- ⚠️ **较复杂** - 需要亚马逊卖家账号或开发者账号
- ⚠️ **审核时间** - 通常需要几天到一周

**数据真实性**：✅ 真实（AI生成的关键词建议）

### 1.2.2 亚马逊API竞争度查询

**实现方式**：
- 使用亚马逊SP-API查询关键词数据
- 获取搜索量、竞争度、转化率等
- 返回结构化的数据

**需要API**：亚马逊SP-API ✅

**数据真实性**：✅ 完全真实（来自亚马逊官方API）

**数据真实性**：✅ 真实（来自API）

### 1.2.5 竞品关键词对比

- 生成对比报告

**需要API**：亚马逊SP-API ✅

**数据真实性**：✅ 真实（来自API）

---

### 1.2.6 长尾关键词挖掘

**实现方式**：
- 使用GPT-3.5生成长尾关键词变体
