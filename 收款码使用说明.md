# 💳 收款码支付使用说明

## 📋 您需要做的事情

### 第一步：保存收款码图片

我已经为您创建了存储目录：`frontend/images/`

**请将您的收款码图片保存到以下位置**：
1. 支付宝收款码 → `frontend/images/alipay_qrcode.png`
2. 微信收款码 → `frontend/images/wechat_qrcode.png`

**图片要求**：
- 格式：PNG 或 JPG
- 文件名必须完全匹配：`alipay_qrcode.png` 和 `wechat_qrcode.png`
- 建议尺寸：300x300像素以上

---

## 🚀 功能说明

### 已实现的功能

1. ✅ **收款码显示**：用户支付时显示您的收款码
2. ✅ **订单管理**：自动生成订单号，保存订单信息
3. ✅ **后台确认**：您可以在后台确认订单
4. ✅ **自动开通会员**：确认后自动开通会员
5. ✅ **自动提醒**：新订单自动发送邮件提醒（可选）

---

## 📱 使用流程

### 用户支付流程

1. 用户选择会员计划 → 点击"立即支付"
2. 选择支付方式（支付宝/微信）
3. 系统显示：
   - 您的收款码
   - 订单号
   - 支付金额
4. 用户扫码支付
5. 用户点击"我已支付"按钮
6. 系统显示"等待管理员确认中..."

### 您的工作流程

1. 收到用户支付（支付宝/微信收款通知）
2. 打开后台管理页面：`http://localhost:5000/admin/orders`
3. 看到待确认订单列表
4. 点击"确认已收款"按钮
5. 系统自动开通会员
6. 用户收到通知：会员已开通

## 🆘 遇到问题？

### 常见问题

1. **收款码不显示**

   - 检查文件名是否完全匹配

2. **无法访问后台**

   - 检查端口是否正确

3. **订单确认失败**
   - 检查订单是否存在
   - 检查用户信息是否正确
