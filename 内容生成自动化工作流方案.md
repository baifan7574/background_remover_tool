# 🤖 内容生成自动化工作流方案

## 📋 核心需求

**问题**：多个平台，每个都要选题、写文案、配标题、配封面、剪视频，时间根本不够。

**解决方案**：**自动化工作流，从选题到内容全部自动生成，你只负责审核和发送。**

---

## 🎯 一、工作流架构设计

### 1.1 完整工作流流程

```
┌─────────────────────────────────────────┐
│  第1步：选题生成（AI自动）              │
│  - 根据产品特点生成20个选题             │

└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│  第2步：内容生成（AI自动）              │
│  - 根据选题生成完整文案                 │
│  - 适配不同平台格式（小红书、知乎等）   │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│  第3步：标题生成（AI自动）              │
│  - 为每个平台生成吸引人的标题           │
│  - 包含关键词、制造冲突、吸引眼球       │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│  第4步：封面生成（AI自动）              │
│  - 自动生成封面图片（小红书、抖音）     │

                    ↓

└─────────────────────────────────────────┘
                    ↓
```

## 🛠️ 二、技术实现方案

### 2.1 工具选择

#### 方案A：使用AI工具（推荐）⭐⭐⭐⭐⭐

**核心工具**：
1. **GPT-4 / Claude**：内容生成（文案、标题、脚本）
2. **Midjourney / DALL-E**：封面图片生成
3. **Runway / Pika**：视频生成（可选）
4. **Python脚本**：自动化工作流

**优势**：
- ✅ 质量高（AI生成的内容质量好）
- ✅ 速度快（几分钟生成一篇内容）
- ✅ 成本低（API费用低）

**劣势**：
- ⚠️ 需要API费用（但很低）
- ⚠️ 需要一些技术基础

---

#### 方案B：使用现成工具（简单）⭐⭐⭐⭐

**核心工具**：
1. **Notion AI**：内容生成
2. **Canva AI**：封面图片生成
3. **剪映AI**：视频生成（可选）
4. **Zapier / Make**：自动化工作流

**优势**：
- ✅ 简单易用（不需要编程）
- ✅ 可视化界面
- ✅ 快速上手

**劣势**：
- ⚠️ 灵活性较低
- ⚠️ 成本可能稍高

---

### 2.2 推荐方案：混合方案

**使用工具**：
- **GPT-4 API**：内容生成（最核心）
- **DALL-E API**：封面图片生成
- **Python脚本**：自动化工作流
- **Canva API**：封面优化（可选）

**成本估算**：
- GPT-4 API：¥0.1-0.3/篇内容
- DALL-E API：¥0.2-0.5/张图片
- **总成本：¥0.3-0.8/篇内容**（非常低）

## 💻 四、完整代码实现

### 4.1 Python工作流脚本

#### 核心代码结构

```python
# content_generator.py
import openai
import json
import os
from datetime import datetime

class ContentGenerator:
    def __init__(self, api_key):
        self.client = openai.OpenAI(api_key=api_key)
        
    def generate_topics(self, count=20):
        """生成选题"""
        prompt = """
        你是一个内容营销专家，专门为跨境电商工具做内容营销。
        
        产品信息：
        - 产品名称：背景移除工具
        - 核心功能：图片背景移除、关键词分析、汇率换算等
        - 价格：19元/月（基础版）
        - 目标用户：个人跨境电商卖家
        
        请生成{count}个内容选题，要求：
        1. 分类：教程类（40%）、对比类（30%）、案例类（20%）、工具介绍类（10%）
        2. 每个选题要吸引眼球，解决用户痛点
        3. 包含关键词：背景移除、跨境电商工具、图片处理
        
        输出格式：JSON格式
        """.format(count=count)
        
        response = self.client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}],
            response_format={"type": "json_object"}
        )
        
        return json.loads(response.choices[0].message.content)
    
    def generate_content(self, topic, platform="小红书"):
        """生成内容"""
        prompts = {
            "小红书": """
            你是一个小红书内容创作者，专门写跨境电商工具相关的内容。
            
            选题：{topic}
            
            请写一篇小红书笔记，要求：
            1. 字数：500-800字
            2. 结构：痛点引入 → 工具介绍 → 操作演示 → 效果对比 → 引导关注
            3. 语言：口语化、亲切、有吸引力
            4. 包含emoji：适当使用emoji增加可读性
            
            输出格式：Markdown格式
            """,
            "知乎": """
            你是一个知乎答主，专门回答跨境电商工具相关的问题。
            
            问题：{topic}
            
            请写一篇知乎回答，要求：
            1. 字数：1,000-1,500字
            2. 结构：问题分析 → 解决方案 → 工具推荐 → 使用建议 → 总结
            3. 语言：专业、详细、有深度
            
            输出格式：Markdown格式
            """,
            "抖音": """
            你是一个抖音视频脚本创作者，专门写跨境电商工具相关的视频脚本。
            
            选题：{topic}
            
            请写一个抖音视频脚本，要求：
            1. 时长：15-60秒
            2. 结构：开头（3秒）→ 内容（30-50秒）→ 结尾（5秒）
            3. 语言：口语化、有节奏感、有吸引力
            
            输出格式：JSON格式
            """
        }
        
        prompt = prompts.get(platform, prompts["小红书"]).format(topic=topic)
        
        response = self.client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}]
        )
        
        return response.choices[0].message.content
    
    def generate_title(self, topic, platform="小红书"):
        """生成标题"""
        prompt = """
        你是一个标题创作专家，专门为内容营销创作吸引人的标题。
        
        内容主题：{topic}
        目标平台：{platform}
        
        请为这个内容生成5个标题，要求：
        1. 包含关键词：背景移除、跨境电商工具
        2. 吸引眼球：制造冲突、悬念、痛点
        3. 适配平台：符合{platform}用户的阅读习惯
        
        输出格式：JSON格式
        """.format(topic=topic, platform=platform)
        
        response = self.client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}],
            response_format={"type": "json_object"}
        )
        
        return json.loads(response.choices[0].message.content)
    
    def generate_cover(self, topic):
        """生成封面（使用DALL-E）"""
        prompt = f"""
        请生成一张小红书笔记封面图片，要求：
        1. 主题：{topic}
        2. 风格：现代、简洁、专业
        3. 包含文字：背景移除、跨境电商工具
        4. 尺寸：1080x1440像素（小红书竖版）
        5. 颜色：主色调蓝色，符合跨境电商工具的专业感
        6. 元素：包含工具图标、数据图表等元素
        """
        
        response = self.client.images.generate(
            model="dall-e-3",
            prompt=prompt,
            size="1024x1024",
            quality="standard",
            n=1
        )
        
        return response.data[0].url
    
    def save_content(self, content_data, output_dir="output"):
        """保存内容"""
        os.makedirs(output_dir, exist_ok=True)
        
        timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
        filename = f"{output_dir}/content_{timestamp}.json"
        
        with open(filename, "w", encoding="utf-8") as f:
            json.dump(content_data, f, ensure_ascii=False, indent=2)
        
        return filename

# 使用示例
if __name__ == "__main__":
    # 初始化
    generator = ContentGenerator(api_key="your-api-key")
    
    # 生成选题
    topics = generator.generate_topics(count=20)
    print("✅ 选题生成完成")
    
    # 为每个选题生成内容
    platforms = ["小红书", "知乎", "抖音"]
    all_content = []
    
    for topic_data in topics["选题列表"]:
        topic = topic_data["标题"]
        content_package = {
            "选题": topic,
            "平台内容": {}
        }
        
        # 为每个平台生成内容
        for platform in platforms:
            # 生成内容
            content = generator.generate_content(topic, platform)
            
            # 生成标题
            titles = generator.generate_title(topic, platform)
            
            # 生成封面（仅小红书和抖音）
            if platform in ["小红书", "抖音"]:
                cover_url = generator.generate_cover(topic)
            else:
                cover_url = None
            
            content_package["平台内容"][platform] = {
                "内容": content,
                "标题": titles,
                "封面": cover_url
            }
        
        all_content.append(content_package)
        print(f"✅ {topic} 内容生成完成")
    
    # 保存所有内容
    output_file = generator.save_content(all_content)
    print(f"✅ 所有内容已保存到：{output_file}")
```
