## 🚀 快速配置步骤

### 步骤1：申请Groq API（5分钟）

1. **访问网站**：https://console.groq.com/
2. **注册账号**：点击"Sign Up"，使用邮箱注册（免费）
3. **创建API Key**：
   - 登录后，点击左侧菜单"API Keys"
   - 点击"Create API Key"
   - 输入名称（如：keyword-analyzer）
   - 点击"Submit"
   - **复制API Key**（只显示一次，请妥善保存）

### 步骤2：配置API Key

在项目根目录创建或编辑`.env`文件：

```env
# Groq API配置（用于关键词提取和长尾关键词挖掘）
GROQ_API_KEY=your_groq_api_key_here
```

**重要**：
- 将`your_groq_api_key_here`替换为你复制的真实API Key
- 确保没有多余的空格
- `.env`文件已经在`.gitignore`中，不会被提交到Git

### 步骤3：安装依赖

```bash
pip install groq pytrends
```

或者使用`requirements.txt`：

```bash
pip install -r requirements.txt
```

### 步骤4：重启服务器

**重启后端服务器**，让配置生效：

```bash
# 停止当前运行的后端服务器（Ctrl+C）
# 然后重新启动
python sk_app.py
```

你应该看到以下消息：
```
✅ Groq API已配置，将使用真实AI提取关键词
✅ Google Trends API已可用，将使用真实趋势数据
```

## 🔍 故障排除

### 问题1：Groq API调用失败

**错误信息**：
```
⚠️ Groq API调用失败: ...
```

**解决方法**：
1. 检查`.env`文件中的`GROQ_API_KEY`是否正确
2. 检查API Key是否有效（可以在Groq控制台重新生成）
3. 检查网络连接
4. 查看服务器日志中的详细错误信息

**如果持续失败**：
- 系统会自动回退到使用模拟数据
- 功能仍然可用，只是数据不是真实的

---

### 问题2：pytrends安装失败

**错误信息**：
```
⚠️ pytrends库未安装
```

**解决方法**：
```bash
pip install pytrends
```

如果仍然失败，可能是因为网络问题，可以尝试：
```bash
pip install pytrends -i https://pypi.tuna.tsinghua.edu.cn/simple
```

---

### 问题3：Google Trends API调用失败

**错误信息**：
```
⚠️ Google Trends API调用失败: ...
```

**可能原因**：
1. 网络连接问题（Google服务可能在某些地区受限）
2. 请求过于频繁（Google Trends有速率限制）

**解决方法**：
- 系统会自动回退到使用模拟数据
- 功能仍然可用，只是趋势数据是模拟的
- 可以稍后重试
