退款政策

重要提示：由于数字服务的特殊性，我们的退款政策较为严格。请在购买前仔细阅读本政策，确保您了解并同意退款条件。购买即表示您已阅读并同意本退款政策。

1. 退款适用范围

本退款政策适用于通过PixelMagic平台购买的所有付费会员服务，包括但不限于：

  • 入门版会员（19元/月）
  • 专业版会员（49元/月）
  • 旗舰版会员（99元/月）
  • 企业版会员（299元/月）
  • 其他付费服务

2. 退款条件（严格限制）

2.1 可申请退款的情况（非常有限）

由于数字服务的特殊性，我们仅在以下极少数情况下考虑退款申请：

  • 服务完全无法使用：因我们的技术故障导致服务完全无法使用超过48小时，且我们无法在合理时间内修复
  • 重复扣款：因系统错误导致重复扣款（需提供支付凭证证明）
  • 严重功能缺失：购买的核心功能完全无法使用，且与描述严重不符（需提供详细证据）

注意：所有退款申请都需要经过严格审核，我们保留最终决定权。

2.2 不可退款的情况（默认情况）

以下情况一律不提供退款，这是我们的默认政策：

  • 已使用服务：购买后已使用任何功能，无论使用次数多少
  • 超过退款期限：购买后超过3天（72小时）
  • 已使用次数：即使未超过3天，但已使用服务1次或以上
  • 误购：因用户误操作购买了错误的套餐（建议在购买前仔细确认）
  • 不满意服务：对服务效果、处理速度、功能等不满意
  • 个人原因：因用户自身原因（如忘记密码、账号被盗、设备问题、网络问题等）导致无法使用服务
  • 违反条款：因违反服务条款导致账号被封禁
  • 正常扣款：免费试用期结束后的正常扣款、自动续费扣款
  • 优惠购买：通过优惠活动、促销活动购买的服务
  • 批量购买：批量购买或企业购买的订单
  • 已获得奖励：已通过邀请好友等方式获得使用次数奖励的用户
  • 其他：任何其他不符合退款条件的情况

3. 退款流程

3.1 申请退款

如需申请退款，请按以下步骤操作：

  1. 登录您的PixelMagic账号
  2. 访问联系我们页面，选择"退款申请"
  3. 填写退款申请表，包括：
    • 订单号或交易号
    • 购买日期
    • 退款原因（详细说明）
    • 联系方式
  4. 提交申请后，我们会在24小时内审核

3.2 退款审核（严格审核）

我们会在收到退款申请后的3-5个工作日内进行严格审核，审核内容包括：

  • 核实订单信息和支付记录
  • 详细检查服务使用情况（使用次数、使用时间、使用功能等）
  • 评估退款原因是否符合退款条件（非常严格）
  • 检查是否存在滥用退款政策的行为
  • 评估用户历史记录（如有多次退款申请，可能被拒绝）

审核标准：我们采用严格的审核标准，大多数退款申请可能被拒绝。审核结果将通过邮件通知您，我们保留最终决定权，不接受申诉。

3.3 退款处理

如果退款申请通过审核：

  • 我们会在3-5个工作日内处理退款
  • 退款将原路返回到您的支付账户
  • 具体到账时间取决于支付平台的处理速度（通常为1-7个工作日）
  • 退款完成后，您的会员服务将立即停止

4. 退款金额（严格限制）

4.1 全额退款（极少情况）

仅在以下极特殊情况下，我们才考虑全额退款：

  • 因我们的技术故障导致服务完全无法使用超过48小时，且我们无法修复
  • 因系统错误导致重复扣款（需提供支付凭证）

注意：即使符合条件，我们也会扣除已使用的服务费用（按使用次数计算）。

4.2 部分退款（基本不提供）

我们通常不提供部分退款。在极少数特殊情况下，如果考虑部分退款：

  • 退款金额 = 支付金额 - 已使用服务费用
  • 已使用服务费用 = 单次使用费用 × 使用次数
  • 单次使用费用 = 会员月费 / 该会员套餐的每日使用次数限制 / 30天

示例：如果基础版会员（19元/月，每天3次背景移除）使用了5次，单次费用约为0.21元，已使用费用为1.05元，退款金额为17.95元。

重要：由于计算复杂且退款成本高，我们通常不提供部分退款。

5. 特殊说明（重要）

5.1 优惠活动（不退款）

通过优惠活动、促销活动、折扣码等方式购买的服务一律不提供退款。优惠购买即表示您已接受不退款条款。

5.2 赠送服务（不退款）

如果您的会员服务包含赠送的使用次数或功能，这些赠送部分不计算在退款金额内，且已使用的赠送服务费用将从退款金额中扣除。

5.3 自动续费（不退款）

自动续费产生的扣款不提供退款。如果您不想继续使用，请在续费前取消自动续费功能。续费后的扣款视为您同意继续使用服务。

5.4 退款次数限制

为防止滥用退款政策，每个用户账号在12个月内最多只能申请1次退款。如果您的账号在12个月内已申请过退款，再次申请将被自动拒绝。

5.5 退款手续费

如果退款申请通过审核，我们可能会收取10%的退款手续费（最低5元），用于支付支付平台手续费和人工审核成本。实际退款金额 = 应退金额 - 退款手续费。
