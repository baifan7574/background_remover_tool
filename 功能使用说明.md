## 🔒 安全建议

### 1. 修改管理员密码

编辑 `sk_app.py` 第190行：
```python
ADMIN_CONFIG = {
    'username': os.getenv('ADMIN_USERNAME', 'admin'),
    'password': os.getenv('ADMIN_PASSWORD', '您的强密码'),  # 修改这里
    'session_key': 'admin_session'
}
```

或者创建 `.env` 文件：
```env
ADMIN_USERNAME=您的用户名
ADMIN_PASSWORD=您的强密码
```

### 2. 修改邮箱授权码

如果邮箱授权码泄露，请：
1. 登录QQ邮箱重新生成授权码
2. 更新 `sk_app.py` 中的配置

## ❓ 常见问题

### Q: 收不到邮件提醒？
A: 检查：
1. QQ邮箱SMTP服务是否已开启
2. 授权码是否正确
3. 查看服务器日志是否有错误

### Q: 无法登录后台？
A: 检查：
1. 用户名和密码是否正确
2. 服务器是否正常运行
3. 查看浏览器控制台是否有错误

### Q: 订单没有自动取消？
A: 检查：
1. 后台定时任务是否启动（查看服务器日志）
2. 订单是否超过24小时
3. 订单状态是否为 `pending`
