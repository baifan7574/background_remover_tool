# 🚀 腾讯云服务器配置 - 一步一步教程

## 📋 目标

1. ✅ 配置好新服务器（腾讯云）
2. ✅ 把网站挂上去
3. ✅ 不再使用旧服务器（PythonAnywhere）

---

## 第1步：获取服务器信息

### 1.1 登录腾讯云控制台

1. 访问：https://console.cloud.tencent.com
2. 登录您的账号
3. 进入"轻量应用服务器"

### 1.2 查看服务器信息

**找到您的服务器，记录以下信息：**
- ✅ **公网 IP 地址**：例如 `123.456.789.0`
- ✅ **登录密码**：在服务器详情页面查看
- ✅ **操作系统**：通常是 Ubuntu 或 CentOS

**把这些信息记下来，后面要用！**

---

## 第2步：连接到服务器

### 2.1 使用 SSH 连接（Windows）

**方法1：使用 PowerShell（推荐）**

1. 按 `Win + X`，选择 "Windows PowerShell"
2. 输入以下命令：
   ```bash
   ssh root@您的公网IP地址
   ```
   例如：`ssh root@123.456.789.0`

3. 第一次连接会提示，输入 `yes`
4. 输入密码（输入时不会显示，直接输入后按回车）

**方法2：使用 Git Bash**

1. 打开 Git Bash
2. 输入同样的命令：
   ```bash
   ssh root@您的公网IP地址
   ```

**如果连接成功，您会看到类似这样的提示：**
```
Welcome to Ubuntu 22.04 LTS
root@server:~#
```

**恭喜！您已经连接到服务器了！** 🎉

---

## 第3步：安装必要软件

### 3.1 更新系统

**在服务器上执行：**
```bash
sudo apt update && sudo apt upgrade -y
```

**等待完成（可能需要几分钟）**

---

### 3.2 安装 Python 和 pip

```bash
sudo apt install python3 python3-pip python3-venv -y
```

**验证安装：**
```bash
python3 --version
pip3 --version
```

**应该显示版本号，说明安装成功！**

---

### 3.3 安装 Nginx（Web 服务器）

```bash
sudo apt install nginx -y
```

**启动 Nginx：**
```bash
sudo systemctl start nginx
sudo systemctl enable nginx
```

**测试：**
- 在浏览器访问：`http://您的公网IP`
- 如果看到 "Welcome to nginx!" 页面，说明安装成功！

---

### 3.4 安装 Git

```bash
sudo apt install git -y
```

**验证：**
```bash
git --version
```

---

## 第4步：上传网站代码

### 4.1 创建项目目录

**在服务器上执行：**
```bash
mkdir -p /var/www/background_remover_tool
cd /var/www/background_remover_tool
```

---

### 4.2 上传代码（两种方法）

**方法1：使用 Git（如果代码在 GitHub）**

```bash
# 在服务器执行
cd /var/www
git clone https://github.com/您的用户名/您的仓库名.git background_remover_tool
cd background_remover_tool
```

**方法2：使用 SCP 从本地上传**

**在本地电脑（Windows PowerShell）执行：**
```bash
# 替换为您的实际路径和 IP
scp -r d:\background_remover_tool\* root@您的公网IP:/var/www/background_remover_tool/
```

**或者使用 FileZilla（图形界面，更简单）：**
1. 下载 FileZilla：https://filezilla-project.org
2. 打开 FileZilla
3. 连接设置：
   - **主机**：您的公网 IP
   - **用户名**：root
   - **密码**：您的服务器密码
   - **端口**：22
4. 连接后，将本地文件拖到服务器 `/var/www/background_remover_tool/` 目录

---

## 第5步：安装 Python 依赖

### 5.1 创建虚拟环境

**在服务器上执行：**
```bash
cd /var/www/background_remover_tool
python3 -m venv venv
source venv/bin/activate
```

**您会看到命令提示符前面出现 `(venv)`，说明虚拟环境已激活！**

---

### 5.2 安装依赖包

```bash
# 确保在虚拟环境中（看到 venv）
pip install -r requirements.txt
```

**如果没有 requirements.txt，安装基本依赖：**
```bash
pip install flask flask-cors pillow supabase python-dotenv rembg
```

**等待安装完成（可能需要几分钟）**

---

## 第6步：配置 Nginx

### 6.1 创建 Nginx 配置文件

```bash
sudo nano /etc/nginx/sites-available/background_remover_tool
```

**在编辑器中输入以下内容：**
```nginx
server {
    listen 80;
    server_name 您的域名.com www.您的域名.com;

    # 静态文件
    location /static {
        alias /var/www/background_remover_tool/static;
    }

    # 上传文件
    location /uploads {
        alias /var/www/background_remover_tool/uploads;
    }

    # 应用
    location / {
        proxy_pass http://127.0.0.1:5000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

**保存并退出：**
- 按 `Ctrl + X`
- 按 `Y` 确认
- 按 `Enter` 保存

**注意：将 `您的域名.com` 替换为您的实际域名！**

---

### 6.2 启用配置

```bash
sudo ln -s /etc/nginx/sites-available/background_remover_tool /etc/nginx/sites-enabled/
sudo nginx -t  # 测试配置是否正确
```

**如果显示 "syntax is ok"，说明配置正确！**

---

### 6.3 重启 Nginx

```bash
sudo systemctl restart nginx
```

---

## 第7步：配置应用服务

### 7.1 创建 systemd 服务文件

```bash
sudo nano /etc/systemd/system/background_remover_tool.service
```

**输入以下内容：**
```ini
[Unit]
Description=Background Remover Tool
After=network.target

[Service]
User=root
WorkingDirectory=/var/www/background_remover_tool
Environment="PATH=/var/www/background_remover_tool/venv/bin"
ExecStart=/var/www/background_remover_tool/venv/bin/python3 app.py
Restart=always

[Install]
WantedBy=multi-user.target
```

**保存并退出：**
- 按 `Ctrl + X`
- 按 `Y`
- 按 `Enter`

---

### 7.2 启动服务

```bash
sudo systemctl daemon-reload
sudo systemctl enable background_remover_tool
sudo systemctl start background_remover_tool
sudo systemctl status background_remover_tool
```

**如果看到 "active (running)"，说明服务启动成功！** ✅

---

## 第8步：配置域名

### 8.1 在 Cloudflare 配置 DNS

1. 登录 Cloudflare：https://dash.cloudflare.com
2. 选择您的域名
3. 进入 "DNS" 设置
4. 添加 A 记录：
   - **类型**：A
   - **名称**：`@`（根域名）
   - **IPv4 地址**：您的腾讯云服务器公网 IP
   - **代理状态**：仅 DNS（灰色云朵）
   - 点击 "保存"

5. 再添加一条 A 记录：
   - **类型**：A
   - **名称**：`www`
   - **IPv4 地址**：相同的 IP
   - **代理状态**：仅 DNS
   - 点击 "保存"

---

### 8.2 配置 SSL 证书（HTTPS）

**在服务器上执行：**
```bash
sudo apt install certbot python3-certbot-nginx -y
sudo certbot --nginx -d 您的域名.com -d www.您的域名.com
```

**按照提示操作：**
- 输入邮箱地址
- 同意服务条款
- 选择是否重定向 HTTP 到 HTTPS（建议选择 2，重定向）

**完成后，您的网站就可以通过 HTTPS 访问了！** 🔒
