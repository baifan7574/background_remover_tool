# ⚡ 部署步骤快速参考（简化版）

## 🎯 快速部署流程

### 第一步：准备服务器（5分钟）

```bash
# 1. 连接服务器
ssh root@your_server_ip

# 2. 更新系统
sudo apt update && sudo apt upgrade -y

# 3. 安装Python和Nginx
sudo apt install python3 python3-pip python3-venv nginx mysql-server -y
```

---

### 第二步：上传代码（5分钟）

```bash
# 在本地压缩代码
tar -czf project.tar.gz background_remover_tool/

# 上传到服务器
scp project.tar.gz root@your_server_ip:/root/

# 在服务器上解压
ssh root@your_server_ip
cd /root
tar -xzf project.tar.gz
cd background_remover_tool
```

---

### 第三步：安装依赖（5分钟）

```bash
# 创建虚拟环境
python3 -m venv venv
source venv/bin/activate

# 安装依赖
pip install Flask flask-cors Pillow rembg numpy opencv-python python-dotenv werkzeug groq pytrends requests
```

---

### 第四步：配置数据库（10分钟）

```bash
# 登录MySQL
sudo mysql -u root -p

# 创建数据库
CREATE DATABASE background_remover_db;
CREATE USER 'app_user'@'localhost' IDENTIFIED BY 'your_password';
GRANT ALL PRIVILEGES ON background_remover_db.* TO 'app_user'@'localhost';
FLUSH PRIVILEGES;
EXIT;
```

---

### 第五步：配置环境变量（5分钟）

```bash
# 创建.env文件
nano .env
```

**添加内容：**
```env
FLASK_ENV=production
DB_HOST=localhost
DB_USER=app_user
DB_PASSWORD=your_password
DB_NAME=background_remover_db
MZFPAY_MERCHANT_ID=10294
MZFPAY_MERCHANT_KEY=X0cJyf2G0EjDKtQe9NMf
```

---

### 第六步：启动服务（5分钟）

```bash
# 测试启动
python3 sk_app.py

# 如果正常，使用systemd创建服务
sudo nano /etc/systemd/system/background-remover.service
```

**服务文件内容：**
```ini
[Unit]
Description=Background Remover Service
After=network.target

[Service]
User=www-data
WorkingDirectory=/root/background_remover_tool
Environment="PATH=/root/background_remover_tool/venv/bin"
ExecStart=/root/background_remover_tool/venv/bin/python sk_app.py

[Install]
WantedBy=multi-user.target
```

**启动服务：**
```bash
sudo systemctl start background-remover
sudo systemctl enable background-remover
```

---

### 第七步：配置Nginx（10分钟）

```bash
sudo nano /etc/nginx/sites-available/background-remover
```

**配置文件内容：**
```nginx
server {
    listen 80;
    server_name your_domain.com;

    location / {
        proxy_pass http://127.0.0.1:5000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

**启用配置：**
```bash
sudo ln -s /etc/nginx/sites-available/background-remover /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx
```

---
