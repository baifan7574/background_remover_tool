### 步骤7: 启动服务器

**方式1: 直接运行**
```bash
python sk_app.py
```

**方式2: 使用Gunicorn**（推荐生产环境）
```bash
pip install gunicorn
gunicorn -w 4 -b 0.0.0.0:5000 sk_app:app
```

**方式3: 使用systemd服务**（推荐）
创建`/etc/systemd/system/background_tool.service`:
```ini
[Unit]
Description=Background Remover Tool
After=network.target

[Service]
User=www-data
WorkingDirectory=/path/to/background_remover_tool
Environment="PATH=/path/to/venv/bin"
ExecStart=/path/to/venv/bin/gunicorn -w 4 -b 0.0.0.0:5000 sk_app:app
Restart=always

[Install]
WantedBy=multi-user.target
```

启动服务:
```bash
sudo systemctl start background_tool
sudo systemctl enable background_tool
```

**问题**: 订单数据存储在内存中（`payment_orders`字典）
