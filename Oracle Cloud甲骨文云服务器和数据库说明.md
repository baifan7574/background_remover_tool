## 📝 具体操作建议

### 步骤1：创建计算实例（服务器）

1. 登录 Oracle Cloud：https://cloud.oracle.com
2. 进入 "Compute" → "Instances"
3. 点击 "Create instance"
4. 选择 "Always Free" 形状（1 OCPU + 1GB 内存）
5. 选择操作系统（推荐 Ubuntu 或 Oracle Linux）
6. 创建实例

**可以创建 2 个实例**

---

### 步骤2：创建数据库

1. 进入 "Databases" → "Autonomous Database"
2. 点击 "Create Autonomous Database"
3. 选择 "Always Free" 选项
4. 配置数据库
5. 创建数据库

**可以创建 2 个数据库**

---

### 步骤3：配置网站托管

**使用 Nginx 反向代理：**
- ✅ 在 1 个实例上安装 Nginx
- ✅ 配置多个虚拟主机
- ✅ 每个虚拟主机对应一个网站
- ✅ 可以托管多个网站

**使用 Docker：**
- ✅ 在实例上安装 Docker
- ✅ 每个网站使用一个容器
- ✅ 可以托管多个网站
