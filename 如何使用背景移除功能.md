# 如何使用背景移除功能

## ✅ 已安装 rembg 库

现在背景移除功能已经可以使用真正的 AI 背景移除了！

---

## 🔄 重要：重启后端服务器

**rembg 库已安装后，必须重启后端服务器才能生效！**

### 方法1：使用启动脚本（推荐）

1. **停止当前运行的后端服务器**
   - 找到运行 `sk_app.py` 的命令窗口
   - 按 `Ctrl+C` 停止服务器

2. **重新运行启动脚本**
   ```bash
   start_dev_servers.bat
   ```

3. **等待服务器启动完成**
   - 后端服务器启动（端口 5000）
   - 前端服务器启动（端口 8000）

### 方法2：手动重启

1. **停止后端服务器**
   - 找到运行 `python sk_app.py` 的命令窗口
   - 按 `Ctrl+C` 停止

2. **重新启动后端服务器**
   ```bash
   python sk_app.py
   ```

3. **保持前端服务器运行**（如果还在运行）
   - 前端服务器（`python -m http.server 8000`）不需要重启

## 🧪 测试背景移除功能

### 步骤1：检查后端日志

启动后端服务器后，查看后端控制台输出：
- ✅ 如果看到 `✅ 使用rembg库进行背景移除...` - 说明 rembg 正常工作
- ❌ 如果看到 `⚠️ rembg库未安装...` - 说明 rembg 未正确导入

### 步骤2：在浏览器测试

1. **打开浏览器**，访问：`http://localhost:8000`

2. **登录账号**（如果还没登录）

3. **点击"背景移除"工具卡片**

4. **上传图片**：
   - 点击上传区域选择图片
   - 或拖拽图片到上传区域

5. **点击"开始处理"按钮**

6. **查看结果**：
   - ✅ 应该看到处理后的图片（背景被移除）
   - ✅ 应该显示"下载图片"按钮
   - ✅ 应该显示使用统计信息

7. **检查后端控制台**：
   - 应该看到 `✅ 使用rembg库进行背景移除...`
   - 应该看到 `✅ 背景移除完成（使用rembg）`

## 🔍 验证 rembg 是否正常工作

### 检查后端控制台输出

处理图片时，后端控制台应该显示：
```
✅ 使用rembg库进行背景移除...
✅ 背景移除完成（使用rembg）
```

如果看到：
```
⚠️ rembg库未安装，使用模拟背景移除（效果不佳）
```

说明 rembg 未正确导入，请检查：
1. rembg 是否真的安装成功
2. 后端服务器是否已重启
3. Python 环境是否正确

### 手动测试 rembg

在命令行运行：
```bash
python -c "from rembg import remove; print('rembg正常')"
```

如果输出 `rembg正常`，说明安装成功。

## 📝 常见问题

### Q1: 处理后图片效果不好怎么办？

**A**: 
- 确保图片质量清晰
- 主体和背景对比明显
- 光线充足的照片效果更好

### Q2: 处理速度很慢？

**A**: 
- 第一次使用会下载 AI 模型（较大，约100MB+）
- 下载完成后，后续处理会快很多
- 大图片处理需要更多时间

### Q3: 提示"rembg库未安装"怎么办？

**A**:
1. 检查安装：`pip show rembg`
2. 重新安装：`pip install rembg`
3. **重启后端服务器**

### Q4: 处理失败或报错？

**A**:
- 检查后端控制台的错误信息
- 确保图片格式正确（JPG、PNG、WEBP等）
- 确保图片大小不超过16MB
