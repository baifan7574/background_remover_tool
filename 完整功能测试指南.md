### 2. 启动前端服务器

**方式1：使用Python简单HTTP服务器**
```bash

cd frontend
python -m http.server 8000
```

**方式2：使用VS Code Live Server**
- 右键点击 `frontend/index.html`
- 选择 "Open with Live Server"

**访问地址：**
```
http://localhost:8000/index.html
```

## 🐛 常见问题排查

### 问题1：后端无法启动

**检查：**
- Python版本（需要3.7+）
- 依赖是否安装：`pip install -r requirements.txt`
- 端口5000是否被占用

### 问题2：前端无法访问

**检查：**
- 前端服务器是否启动
- 端口8000是否被占用
- 浏览器控制台是否有错误

### 问题3：API请求失败

**检查：**
- 后端服务器是否运行
- CORS配置是否正确
- 请求URL是否正确
- 浏览器控制台错误信息

### 问题4：支付订单创建失败

**检查：**
- 后端日志中的错误信息
- 码支付配置是否正确
- 网络连接是否正常

### 问题5：使用次数不扣除

**检查：**
- 后端日志：`记录使用次数已记录`
- `data/profiles.json` 中的 `daily_usage` 是否更新
- 用户是否登录
