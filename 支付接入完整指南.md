## 💰 费用说明

### 支付宝费用
- **入驻费用**：免费
- **服务费率**：
  - 电脑网站支付：0.6%
  - 手机网站支付：0.6%
  - 当面付（扫码支付）：0.6%
- **提现费用**：免费（T+1到账）
- **最低费率**：无最低收费
- **结算周期**：T+1（交易成功后第二天到账）

### 微信支付费用
- **入驻费用**：免费
- **服务费率**：
  - Native支付（扫码）：0.6%
  - JSAPI支付（公众号/小程序）：0.6%
  - H5支付（手机浏览器）：0.6%
- **提现费用**：免费（T+1到账）
- **最低费率**：无最低收费
- **结算周期**：T+1（交易成功后第二天到账）

### 总体成本
- **无入驻费用**：两个平台都免费入驻
- **交易手续费**：每笔交易收取0.6%手续费
  - 例如：用户支付100元，实际到账99.4元，手续费0.6元
- **提现免费**：资金自动结算到对公账户，无需额外费用
- **年费/月费**：无

### 成本示例
假设月收入10,000元：
- 支付宝手续费：10,000 × 0.6% = 60元
- 微信支付手续费：10,000 × 0.6% = 60元
- **总手续费**：120元/月
- **实际到账**：9,880元/月

## 🔵 支付宝接入步骤

### 1. 注册支付宝开放平台账号
1. 访问：https://open.alipay.com/
2. 使用企业账号注册（个人账号无法申请支付功能）
3. 完成企业认证（需要营业执照）

### 2. 创建应用
1. 登录支付宝开放平台
2. 进入"控制台" → "网页&移动应用" → "创建应用"
3. 选择应用类型：
   - **电脑网站支付**：适用于PC端
   - **手机网站支付**：适用于移动端H5
   - **当面付**：适用于扫码支付
4. 填写应用信息：
   - 应用名称：AI背景移除工具
   - 应用图标：上传Logo
   - 应用描述：AI智能背景移除服务

### 3. 配置应用信息
1. **添加功能**：
   - 电脑网站支付（alipay.trade.page.pay）
   - 手机网站支付（alipay.trade.wap.pay）
   - 当面付（alipay.trade.precreate）

2. **配置回调地址**：
   - 异步通知地址：`https://yourdomain.com/api/payment/alipay/notify`
   - 同步返回地址：`https://yourdomain.com/payment/success`

3. **配置应用网关**：
   - 网关地址：`https://openapi.alipay.com/gateway.do`
   - 沙箱网关：`https://openapi.alipaydev.com/gateway.do`（测试用）

### 4. 生成密钥
1. 下载支付宝密钥生成工具：https://opendocs.alipay.com/common/02kkv7
2. 生成RSA2密钥对：
   - 应用私钥（app_private_key）
   - 应用公钥（app_public_key）
3. 上传应用公钥到支付宝开放平台
4. 获取支付宝公钥（alipay_public_key）

### 5. 获取应用信息
- **AppID**：在应用详情页面查看
- **应用私钥**：刚才生成的私钥
- **支付宝公钥**：在应用详情页面下载

### 6. 签约产品
1. 进入"产品中心"
2. 签约以下产品：
   - 电脑网站支付
   - 手机网站支付
   - 当面付
3. 等待审核（通常1-3个工作日）

### 7. 上线应用
1. 提交应用审核
2. 等待审核通过（通常3-7个工作日）
3. 审核通过后即可使用正式环境

## 🟢 微信支付接入步骤

### 1. 注册微信支付商户平台
1. 访问：https://pay.weixin.qq.com/
2. 使用企业账号注册（需要营业执照）
3. 完成企业认证

### 2. 申请支付产品
1. 登录微信支付商户平台
2. 进入"产品中心" → "我的产品"
3. 申请以下产品：
   - **Native支付**：扫码支付
   - **JSAPI支付**：公众号/小程序支付
   - **H5支付**：手机浏览器支付

### 3. 配置支付参数
1. **获取商户信息**：
   - 商户号（mch_id）：在"账户中心"查看
   - API密钥（api_key）：在"账户中心" → "API安全"设置

2. **配置支付授权目录**：
   - 支付授权目录：`https://yourdomain.com/payment/`
   - 测试授权目录：`https://yourdomain.com/payment/`

3. **配置回调地址**：
   - 支付结果通知URL：`https://yourdomain.com/api/payment/wechat/notify`

### 4. 配置API证书（可选，推荐）
1. 进入"账户中心" → "API安全" → "API证书"
2. 下载证书文件：
   - `apiclient_cert.pem`（证书）
   - `apiclient_key.pem`（私钥）
3. 用于退款等高级功能（当前代码使用MD5签名，可暂不配置）

### 5. 获取应用信息
- **AppID**：需要关联微信公众号或小程序
  - 如果使用公众号：在微信公众平台获取
  - 如果使用小程序：在微信公众平台获取
  - 如果只做扫码支付：可以使用商户平台的AppID
- **商户号（mch_id）**：在商户平台查看
- **API密钥（api_key）**：在商户平台设置

### 6. 关联公众号/小程序（如需要）
1. 如果使用JSAPI支付，需要：
   - 注册微信公众号或小程序
   - 在微信支付商户平台关联公众号/小程序
   - 获取AppID和AppSecret

### 7. 提交审核
1. 完成所有配置后提交审核
2. 等待审核通过（通常1-3个工作日）

## 🔔 回调配置

### 支付宝回调配置

#### 回调地址
```
https://yourdomain.com/api/payment/alipay/notify
```

#### 回调处理流程（已在代码中实现）
1. 接收支付宝POST请求
2. 验证签名（`verify_notify`方法）
3. 检查交易状态（`TRADE_SUCCESS`或`TRADE_FINISHED`）
4. 更新订单状态为`paid`
5. 调用`activate_membership`自动开通会员
6. 返回`success`给支付宝

#### 配置要求
- **必须使用HTTPS**：生产环境必须配置SSL证书
- **必须公网可访问**：不能是localhost或内网地址
- **必须返回success**：处理成功后必须返回字符串`success`

### 微信支付回调配置

#### 回调地址
```
https://yourdomain.com/api/payment/wechat/notify
```

#### 回调处理流程（已在代码中实现）
1. 接收微信POST的XML数据
2. 验证签名（`verify_notify`方法）
3. 检查交易状态（`trade_state == 'SUCCESS'`）
4. 更新订单状态为`paid`
5. 调用`activate_membership`自动开通会员
6. 返回XML格式的成功响应

#### 配置要求
- **必须使用HTTPS**：生产环境必须配置SSL证书
- **必须公网可访问**：不能是localhost或内网地址
- **必须返回正确格式**：返回`{'code': 'SUCCESS', 'message': '处理成功'}`

## ⚙️ 自动开通会员流程

### 完整流程
```
用户支付 → 支付平台回调 → 验证签名 → 更新订单状态 → 自动开通会员
```

### 代码实现位置

#### 1. 支付宝回调处理
**文件**：`backend/payment_api.py`
**函数**：`alipay_notify()`
**关键代码**：
```python
# 更新订单状态
update_result = order_manager.update_order_status(
    order_no=order_no,
    status='paid',
    transaction_id=trade_no,
    payment_data=data
)

# 激活会员
if update_result['success']:
    activate_result = order_manager.activate_membership(order_no)
```

#### 2. 微信支付回调处理
**文件**：`backend/payment_api.py`
**函数**：`wechat_notify()`
**关键代码**：
```python
# 更新订单状态
update_result = order_manager.update_order_status(
    order_no=order_no,
    status='paid',
    transaction_id=transaction_id,
    payment_data=notify_data
)

# 激活会员
if update_result['success']:
    activate_result = order_manager.activate_membership(order_no)
```

#### 3. 会员激活逻辑
**文件**：`backend/order_manager.py`
**函数**：`activate_membership()`
**功能**：
- 根据订单信息获取会员计划
- 计算会员到期时间
- 更新`user_profiles`表的会员信息
- 记录会员激活日志

## 🔐 环境变量配置

### 创建`.env`文件
在项目根目录创建`.env`文件，配置以下环境变量：

```bash
# 支付宝配置
ALIPAY_APP_ID=你的支付宝AppID
ALIPAY_PRIVATE_KEY=-----BEGIN RSA PRIVATE KEY-----
你的应用私钥内容
-----END RSA PRIVATE KEY-----
ALIPAY_PUBLIC_KEY=-----BEGIN PUBLIC KEY-----
你的应用公钥内容
-----END PUBLIC KEY-----
ALIPAY_ALIPAY_PUBLIC_KEY=-----BEGIN PUBLIC KEY-----
支付宝公钥内容
-----END PUBLIC KEY-----

# 微信支付配置
WECHAT_APP_ID=你的微信AppID
WECHAT_MCH_ID=你的商户号
WECHAT_API_KEY=你的API密钥
WECHAT_NOTIFY_URL=https://yourdomain.com/api/payment/wechat/notify

# 环境配置
FLASK_ENV=production  # production或development
SUPABASE_URL=你的Supabase URL
SUPABASE_KEY=你的Supabase Key

# 域名配置（用于回调地址）
DOMAIN=https://yourdomain.com
```

### 密钥格式说明

#### 支付宝私钥格式
```
-----BEGIN RSA PRIVATE KEY-----
MIIEvgIBADANBgkqhkiG9w0BAQEFAASCBKgwggSkAgEAAoIBAQC...
（多行内容）
-----END RSA PRIVATE KEY-----
```

#### 支付宝公钥格式
```
-----BEGIN PUBLIC KEY-----
MIIBIjANBgkqhkiG9w0BAQEFAAOCAQ8AMIIBCgKCAQEA...
（多行内容）
-----END PUBLIC KEY-----
```

### 安全建议
1. **不要将`.env`文件提交到Git**：添加到`.gitignore`
2. **使用环境变量管理**：生产环境使用服务器环境变量
3. **定期更换密钥**：建议每6个月更换一次API密钥
4. **限制密钥权限**：只给必要的服务器访问权限

## ❓ 常见问题

### Q1: 回调地址无法访问怎么办？
**A**: 
1. 确保服务器配置了HTTPS
2. 确保回调地址公网可访问
3. 检查防火墙是否开放443端口
4. 使用`curl`测试回调地址是否可访问

### Q2: 签名验证失败怎么办？
**A**:
1. 检查密钥是否正确配置
2. 确认使用的是RSA2签名（支付宝）
3. 检查密钥格式是否正确（包含BEGIN和END标记）
4. 确认环境变量是否正确加载

### Q3: 会员没有自动开通怎么办？
**A**:
1. 检查回调是否正常接收（查看日志）
2. 检查订单状态是否更新为`paid`
3. 检查`activate_membership`函数是否执行
4. 查看数据库`user_profiles`表是否更新
5. 检查`membership_logs`表是否有记录

### Q4: 如何查看支付回调日志？
**A**:
1. 查看Flask应用日志
2. 检查`payment_api.py`中的`current_app.logger`输出
3. 在回调函数中添加详细日志记录

### Q5: 测试环境和生产环境如何切换？
**A**:
1. 设置环境变量`FLASK_ENV=development`（测试）或`production`（生产）
2. 支付宝会自动切换到沙箱网关
3. 微信支付需要手动切换沙箱URL

### Q6: 手续费如何计算？
**A**:
- 每笔交易收取0.6%手续费
- 手续费在结算时自动扣除
- 实际到账金额 = 支付金额 × (1 - 0.006)

### Q7: 资金什么时候到账？
**A**:
- T+1结算：交易成功后第二天到账
- 自动结算到对公账户
- 无需手动提现

### Q8: 需要营业执照吗？
**A**:
- **支付宝**：需要企业认证，必须提供营业执照
- **微信支付**：需要企业认证，必须提供营业执照
- 个人账号无法申请支付功能

### Q9: 审核需要多长时间？
**A**:
- 支付宝应用审核：3-7个工作日
- 微信支付审核：1-3个工作日
- 产品签约审核：1-3个工作日

### Q10: 可以只接入一个支付方式吗？
**A**:
- 可以，代码已经支持单独接入
- 如果只接入支付宝，微信支付相关代码不会执行
- 建议同时接入两个，提高用户支付成功率
