## 📋 部署步骤

### 1. 上传文件
```
background_remover_tool/
├── sk_app.py
├── data_manager.py
├── requirements.txt
├── .env (可选)
├── frontend/
└── data/ (会自动创建)
```

### 2. 安装依赖
```bash
pip install -r requirements.txt
```

### 3. 创建数据目录
```bash
mkdir -p data && chmod 755 data
```

### 4. 启动服务器
```bash
# 测试
python sk_app.py

# 生产环境（推荐）
gunicorn -w 4 -b 0.0.0.0:5000 sk_app:app
```
