### 1. GPT关键词提取替代方案

#### 方案1.1：使用Hugging Face Inference API（免费）⭐

**申请难度**：⭐⭐（简单）

**优点**：
- ✅ 免费额度：30,000次/月
- ✅ 申请简单，只需注册账号
- ✅ 支持多种模型（如BART、T5等）
- ✅ 可以用于关键词提取

**申请步骤**：
1. 访问：https://huggingface.co/
2. 注册账号（免费）
3. 获取API Token：Settings → Access Tokens → New Token
4. 选择权限：Read（读取模型）

**API示例**：
```python
import requests

def extract_keywords_huggingface(text):
    API_URL = "https://api-inference.huggingface.co/models/facebook/bart-large-cnn"
    headers = {"Authorization": f"Bearer YOUR_TOKEN"}
    
    response = requests.post(API_URL, headers=headers, json={
        "inputs": text,
        "parameters": {"max_length": 200, "min_length": 50}
    })
    return response.json()
```

**免费额度**：30,000次/月

**申请难度**：⭐⭐（简单）

**申请步骤**：
1. 访问：https://console.groq.com/
2. 注册账号（免费）
3. 创建API Key：API Keys → Create API Key
4. 复制API Key

**API示例**：
```python
from groq import Groq

client = Groq(api_key="YOUR_API_KEY")

def extract_keywords_groq(text):
    chat_completion = client.chat.completions.create(
        messages=[
            {"role": "system", "content": "你是一个关键词提取专家，从产品描述中提取关键词。"},
            {"role": "user", "content": f"请提取以下产品描述的关键词：{text}"}
        ],
        model="mixtral-8x7b-32768",
        temperature=0.7
    )
    return chat_completion.choices[0].message.content
```

**免费额度**：14,400次/天

**安装步骤**：

2. 安装并运行
3. 下载模型：`ollama pull llama2` 或 `ollama pull mistral`
4. 在代码中调用本地API

**API示例**：
```python
import requests

def extract_keywords_ollama(text):
    response = requests.post("http://localhost:11434/api/generate", json={
        "model": "llama2",
        "prompt": f"请从以下产品描述中提取关键词：{text}",
        "stream": False
    })
    return response.json()["response"]
```

**优点**：
- ✅ 完全免费
- ✅ 无使用限制
- ✅ 数据安全

**缺点**：
- ⚠️ 需要本地运行（占用CPU/GPU资源）

```python
pytrend = TrendReq(hl='en-US', tz=360)

def get_keyword_trends(keywords):

    trend_data = pytrend.interest_over_time()
    return trend_data
```
