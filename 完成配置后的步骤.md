
**应该看到：**

---

### 步骤2：启动后端服务

```bash
sudo systemctl start background_remover_tool
```

---

### 步骤3：检查服务状态

```bash
sudo systemctl status background_remover_tool
```

**应该看到：**
```
Active: active (running)
```

**如果显示 `inactive` 或 `failed`，查看日志：**
```bash
sudo journalctl -u background_remover_tool -n 50 --no-pager
```

---

### 步骤4：设置开机自启

```bash
sudo systemctl enable background_remover_tool
```

**这样服务器重启后，服务会自动启动。**

---
