## 🔧 切换步骤（5分钟完成）

### 步骤1：注册虎皮椒账号

1. 访问：https://www.hupijiao.com
2. 点击"注册" → 选择"个人注册"
3. 填写信息并完成实名认证
4. 登录后台

### 步骤2：创建商户并获取配置

1. 登录后，进入"商户中心"
2. 创建新商户（或使用默认商户）
3. 获取以下信息：
   - **商户ID**（PID）
   - **商户密钥**（Key）
   - **接口地址**：通常是 `https://www.hupijiao.com/api/create`

### 步骤3：修改代码配置

**文件**：`sk_app.py`

找到这部分代码（大约第87-88行）：

```python
MZFPAY_MERCHANT_ID = '10294'
MZFPAY_MERCHANT_KEY = 'X0cJyf2G0EjDKtQe9NMf'
```

**修改为**：

```python
MZFPAY_MERCHANT_ID = '你的虎皮椒商户ID'
MZFPAY_MERCHANT_KEY = '你的虎皮椒商户密钥'
```

**注意**：虎皮椒使用标准易支付接口，代码不需要改动，只需修改这两行配置！

### 步骤4：修改接口地址（可选）

如果虎皮椒的接口地址不同，需要修改：

**文件**：`backend/mzfpay_client.py`

找到第24-25行：

```python
self.api_url = "https://pay.mzfpay.com/xpay/epay/mapi.php"
self.submit_url = "https://pay.mzfpay.com/xpay/epay/submit.php"
```

**修改为虎皮椒的接口地址**（在虎皮椒后台查看具体地址）：

```python
self.api_url = "https://www.hupijiao.com/api/create"  # 或虎皮椒提供的API地址
self.submit_url = "https://www.hupijiao.com/api/submit"  # 或虎皮椒提供的提交地址
```

**注意**：大多数情况下，虎皮椒的接口地址是标准的，可能不需要修改。先测试，如果失败再修改。

### 步骤5：测试支付

1. **重启后端服务器**
   - 关闭当前服务器（Ctrl+C）
   - 重新运行：`python sk_app.py` 或双击 `启动后端服务器.bat`

2. **运行诊断工具**
   ```bash
   python 支付问题诊断工具.py
   ```

3. **测试支付流程**
   - 打开浏览器：http://localhost:8000
   - 登录账号
   - 选择会员计划 → 点击"立即升级"
   - 选择支付方式 → 点击"确认支付"
   - 查看是否能正常跳转到支付页面

## 🔍 如果遇到问题

### 问题1：接口地址不对

**症状**：创建订单失败，返回"接口不存在"或"404错误"

**解决**：
1. 登录虎皮椒后台
2. 查看API文档，确认正确的接口地址
3. 修改 `backend/mzfpay_client.py` 中的接口地址

### 问题2：签名验证失败

**症状**：返回"签名错误"

**解决**：
1. 确认商户ID和密钥是否正确
2. 确认密钥没有多余空格
3. 运行诊断工具检查签名生成

### 问题3：支付页面无法打开

**症状**：支付链接生成成功，但点击后无法打开

**解决**：
1. 检查支付链接是否正确
2. 联系虎皮椒客服确认商户状态
3. 检查是否需要在后台配置回调地址
