## 🔍 API/服务成本分析

### 1. **Groq API（关键词提取）**

**定价信息：**
- 免费额度：每天 **14,400 请求**（免费）
- 超过免费额度：约 **$0.27/100万tokens**
- 我们的使用：每次关键词提取约 **500-1000 tokens**

**成本计算：**
- 每次关键词提取成本：**≈ $0.0001-0.0002**（约 ¥0.0007-0.0014）
- 免费会员每天 **10次** 关键词分析：**≈ ¥0.007-0.014/天**
- **结论：✅ 成本极低，免费额度足够覆盖大量用户**

### 2. **Google Trends API (pytrends)**

**定价信息：**
- **完全免费**（非官方API）
- 限制：每个IP每小时约 **5个查询**
- 我们的使用：关键词趋势分析

**成本计算：**
- **成本：¥0**（免费）
- **结论：✅ 零成本，但需要注意频率限制**

### 3. **ExchangeRate-API（汇率换算）**

**定价信息：**
- 免费版：每月 **1,500 请求**
- 付费版：$9.99/月（无限请求）
- 我们的使用：每次汇率换算 **1次请求**

**成本计算：**
- 免费会员每天 **10次** 汇率换算 = **300次/月**
- 100个免费会员 = **30,000次/月**（超出免费额度）
- 需要付费版：**$9.99/月 ≈ ¥70/月**
- **结论：⚠️ 需要监控使用量，建议限制免费用户或寻找替代方案**

### 4. **rembg（背景移除）**

**定价信息：**
- **本地处理，无API成本**
- 仅消耗服务器CPU/内存资源

**成本计算：**
- 服务器资源成本：**极低**（单次处理约几秒）
- 1000次处理 ≈ **0.1-0.5小时CPU时间**
- **结论：✅ 成本极低，可忽略**

### 5. **其他图片处理工具**

**定价信息：**
- 图片压缩、格式转换、裁剪、水印：**本地处理**
- 使用 Pillow 库（Python），无API成本

**成本计算：**
- **成本：¥0**（仅服务器资源）
- **结论：✅ 零API成本**

## 🎁 邀请奖励成本核算

### **邀请奖励使用次数**

**老用户奖励：**
- 每成功邀请1人：**+50次免费使用**（一次性，30天内有效）
- 假设：每次使用平均 **2次API调用**（关键词分析1次 + 汇率换算1次）
- 50次 × 2次API = **100次API调用**

**新用户奖励：**
- 注册成功：**+20次免费使用**（一次性，30天内有效）

## 🚨 成本风险分析

### **主要成本来源：**

1. **ExchangeRate-API** - 占比 **98%** 的成本
   - 免费版：仅1,500次/月
   - 100个免费用户 = 30,000次/月（需要20倍付费额度）
   - 成本：¥70/月（付费版）

2. **Groq API** - 占比 **2%** 的成本
   - 免费额度充足（14,400次/天）
   - 成本极低

### **风险点：**

1. **ExchangeRate-API超出免费额度**
   - 免费版只能支持 **5个免费会员**（10次/天 × 30天 = 300次）
   - 超过5个用户就需要付费版（¥70/月）

2. **用户增长导致成本激增**
   - 每个免费会员每月成本 **¥14.1**
   - 1000个免费会员 = **¥14,100/月**（不可持续）
