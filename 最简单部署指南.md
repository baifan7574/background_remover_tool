## 🚀 部署步骤（最简单方式）

### 方式1：使用 PythonAnywhere（推荐，免费）

#### 第1步：注册/登录 PythonAnywhere
1. 访问：https://www.pythonanywhere.com
2. 注册账号（如果还没有）
3. 登录

#### 第2步：上传代码
1. 登录后，点击顶部 **"Files"** 标签
2. 进入 `/home/你的用户名/mysite/` 目录
3. 点击 **"Upload a file"** 按钮
4. 上传这些文件：
   - `sk_app.py` （最重要的，这是您的网站代码）
   - `data_manager.py`
   - `supabase_db.py`
   - `requirements.txt`
   - `frontend/` 整个文件夹（包含所有前端文件）

#### 第3步：安装依赖
1. 点击顶部 **"Consoles"** 标签
2. 点击 **"Start a new console"**
3. 选择 **"Bash"**
4. 在控制台输入：
```bash
cd ~/mysite
pip3.10 install --user flask flask-cors pillow supabase python-dotenv rembg werkzeug requests
```

#### 第4步：配置数据库连接
1. 在 **"Files"** 标签，创建文件 `.env`
2. 在里面写：
```
SUPABASE_URL=你的Supabase地址
SUPABASE_KEY=你的Supabase密钥
SUPABASE_SERVICE_KEY=你的Service密钥
```

#### 第5步：配置Web应用
1. 点击顶部 **"Web"** 标签
2. 点击 **"Add a new web app"**
3. 选择 **"Flask"** → 选择 **"Python 3.10"**
4. 填写路径：`/home/你的用户名/mysite/sk_app.py`
5. 点击 **"Next"** → **"Finish"**

#### 第6步：设置静态文件
1. 在 **"Web"** 标签找到 **"Static files"**
2. 添加：
   - URL: `/static/`
   - Directory: `/home/你的用户名/mysite/frontend/`
3. 点击 **"Save"**

#### 第7步：重启应用
1. 在 **"Web"** 标签
2. 点击绿色的 **"Reload 你的域名.pythonanywhere.com"** 按钮

#### 完成！
现在您的网站可以通过以下地址访问：
- `https://你的用户名.pythonanywhere.com`
