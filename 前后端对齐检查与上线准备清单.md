## 🚀 六、上线部署步骤

### 步骤1：支付网关接入

```bash
# 1. 安装支付SDK
pip install alipay-sdk-python
pip install wechatpay-python

# 2. 配置支付参数
# 编辑 backend/payment_config.py
# 填入真实的商户ID、密钥等

# 3. 测试支付流程
python test_payment.py
```

### 步骤2：代码部署

```bash
# 1. 检查依赖
pip install -r requirements.txt

# 2. 配置环境变量
export FLASK_ENV=production
export SECRET_KEY=your_secret_key

# 3. 启动服务
python sk_app.py
# 或使用 gunicorn
gunicorn -w 4 -b 0.0.0.0:5000 sk_app:app
```

### 步骤3：服务器配置

1. **配置反向代理（Nginx）**
   ```nginx
   server {
       listen 80;
       server_name yourdomain.com;
       
       location / {
           proxy_pass http://127.0.0.1:5000;
           proxy_set_header Host $host;
           proxy_set_header X-Real-IP $remote_addr;
       }
       
       location /static {
           alias /path/to/frontend;
       }
   }
   ```

2. **配置SSL证书**
   ```bash
   certbot --nginx -d yourdomain.com
   ```

3. **配置防火墙**
   ```bash
   ufw allow 80/tcp
   ufw allow 443/tcp
   ufw allow 5000/tcp  # 仅允许本地访问
   ```
