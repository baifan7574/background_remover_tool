## 第2步：配置老域名

### 2.1 在域名服务商配置 DNS

**在 Cloudflare（或其他域名服务商）：**

1. 登录 Cloudflare
2. 选择您的域名
3. 进入 DNS 设置
4. 添加 A 记录：
   - **类型**：A
   - **名称**：`@`（根域名）或 `www`
   - **IPv4 地址**：您的腾讯云服务器公网 IP
   - **代理状态**：仅 DNS（灰色云朵）
   - 点击"保存"

---

### 2.2 在服务器上配置 Nginx

**创建 Nginx 配置文件：**
```bash
sudo nano /etc/nginx/sites-available/background_remover_tool
```

**配置文件内容：**
```nginx
server {
    listen 80;
    server_name 您的域名.com www.您的域名.com;

    # 静态文件
    location /static {
        alias /var/www/background_remover_tool/static;
    }

    # 上传文件
    location /uploads {
        alias /var/www/background_remover_tool/uploads;
    }

    # 应用
    location / {
        proxy_pass http://127.0.0.1:5000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

**启用配置：**
```bash
sudo ln -s /etc/nginx/sites-available/background_remover_tool /etc/nginx/sites-enabled/
sudo nginx -t  # 测试配置
sudo systemctl restart nginx  # 重启 Nginx
```

## 第3步：本地测试功能

### 3.1 在服务器上运行应用

**创建 systemd 服务：**
```bash
sudo nano /etc/systemd/system/background_remover_tool.service
```

**服务文件内容：**
```ini
[Unit]
Description=Background Remover Tool
After=network.target

[Service]
User=root
WorkingDirectory=/var/www/background_remover_tool
Environment="PATH=/var/www/background_remover_tool/venv/bin"
ExecStart=/var/www/background_remover_tool/venv/bin/python3 app.py
Restart=always

[Install]
WantedBy=multi-user.target
```

**启动服务：**
```bash
sudo systemctl daemon-reload
sudo systemctl enable background_remover_tool
sudo systemctl start background_remover_tool
sudo systemctl status background_remover_tool  # 查看状态
```

## 第4步：实现 Git 自动部署

### 4.1 在服务器上配置 Git Webhook

**创建部署脚本：**
```bash
sudo nano /var/www/background_remover_tool/deploy.sh
```

**部署脚本内容：**
```bash
#!/bin/bash
cd /var/www/background_remover_tool

# 拉取最新代码
git pull origin main

# 激活虚拟环境
source venv/bin/activate

# 安装新依赖
pip install -r requirements.txt

# 重启服务
sudo systemctl restart background_remover_tool

echo "✅ 部署完成"
```

**设置执行权限：**
```bash
chmod +x /var/www/background_remover_tool/deploy.sh
```

---

### 4.2 配置 GitHub Webhook（推荐）

**在 GitHub 仓库：**
1. 进入 Settings → Webhooks
2. 点击 "Add webhook"
3. 配置：
   - **Payload URL**：`http://您的域名.com/webhook`（需要创建 webhook 端点）
   - **Content type**：application/json
   - **Secret**：设置一个密钥
   - **Events**：选择 "Just the push event"
   - 点击 "Add webhook"

---

### 4.3 在应用中添加 Webhook 端点

**在您的 Flask 应用中添加：**
```python
import hmac
import hashlib
from flask import request, jsonify

@app.route('/webhook', methods=['POST'])
def webhook():
    # 验证 GitHub Webhook
    secret = '您的密钥'
    signature = request.headers.get('X-Hub-Signature-256')
    
    if signature:
        hash_object = hmac.new(secret.encode(), request.data, hashlib.sha256)
        expected_signature = 'sha256=' + hash_object.hexdigest()
        
        if hmac.compare_digest(signature, expected_signature):
            # 执行部署
            import subprocess
            subprocess.run(['/var/www/background_remover_tool/deploy.sh'], shell=True)
            return jsonify({'status': 'success'}), 200
    
    return jsonify({'status': 'unauthorized'}), 401
```

### 4.4 使用 GitHub Actions（更简单的方法）

**创建 GitHub Actions 工作流：**
```bash
# 在本地项目创建
mkdir -p .github/workflows
nano .github/workflows/deploy.yml
```

**工作流文件内容：**
```yaml
name: Deploy to Tencent Cloud

on:
  push:
    branches: [ main ]

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
    - name: Deploy to server
      uses: appleboy/ssh-action@master
      with:
        host: ${{ secrets.HOST }}
        username: ${{ secrets.USERNAME }}
        key: ${{ secrets.SSH_KEY }}
        script: |
          cd /var/www/background_remover_tool
          git pull origin main
          source venv/bin/activate
          pip install -r requirements.txt
          sudo systemctl restart background_remover_tool
```

**在 GitHub 仓库设置 Secrets：**
1. 进入 Settings → Secrets and variables → Actions
2. 添加：
   - `HOST`：您的服务器公网 IP
   - `USERNAME`：root
   - `SSH_KEY`：您的 SSH 私钥

### 4.5 使用 Git Hooks（最简单的方法）

**在服务器上配置：**
```bash
cd /var/www/background_remover_tool
git config receive.denyCurrentBranch ignore

# 创建 post-receive hook
cat > .git/hooks/post-receive << 'EOF'
#!/bin/bash
cd /var/www/background_remover_tool
git checkout -f
source venv/bin/activate
pip install -r requirements.txt
sudo systemctl restart background_remover_tool
EOF

chmod +x .git/hooks/post-receive
```

**在本地配置 Git remote：**
```bash
git remote add tencent ssh://root@您的公网IP/var/www/background_remover_tool/.git
```

**推送代码：**
```bash
git push tencent main
```
