# 🚀 快速启动指南

## ✅ 后端服务器已启动！

我已经帮您启动了后端服务器（端口5000）。

---

## 📋 现在需要做的

### 第一步：启动前端服务器

**打开一个新的命令行窗口**，然后输入：

```bash
cd D:\background_remover_tool\frontend
python -m http.server 8000
```

**或者直接双击：** `启动前端服务器.bat` 文件

---

### 第二步：访问网站

**在浏览器打开：** `http://localhost:8000`

---

## ✅ 检查服务器状态

- **端口**：5000
- **健康检查**：访问 `http://localhost:5000/health`
- **应该看到**：JSON格式的健康状态信息

### 前端服务器（需要您启动）
- **端口**：8000
- **访问地址**：`http://localhost:8000`
