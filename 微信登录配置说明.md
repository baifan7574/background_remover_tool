# 微信登录配置说明

## 📋 功能说明

已实现真实的微信扫码登录功能，用户扫描二维码后可以自动注册并登录。

## 🔧 配置步骤

### 1. 注册微信开放平台账号

1. 访问 [微信开放平台](https://open.weixin.qq.com/)
2. 注册并登录账号
3. **开发者资质认证**（必需）：
   - 需要缴纳 **300元人民币** 的认证费用（一次性费用）
   - 企业认证需要提交加盖公章的营业执照
   - 个人开发者也需要完成认证
   - 认证通过后才能创建应用并获取 AppID
4. 创建网站应用

### 2. 获取 AppID 和 AppSecret

1. 在微信开放平台控制台，进入"网站应用"
2. 创建新应用或使用已有应用
3. 获取 **AppID** 和 **AppSecret**

### 3. 配置授权回调域名

在微信开放平台中，设置授权回调域名为：
```
您的域名（例如：yourdomain.com）
```

**注意**：不需要加 `http://` 或 `https://`，只需要域名本身。

### 4. 配置环境变量

在服务器的 `.env` 文件中添加：

```bash
# 微信开放平台配置
WECHAT_APPID=您的AppID
WECHAT_APPSECRET=您的AppSecret
```

### 5. 安装依赖

在服务器上执行：

```bash
cd /var/www/background_remover_tool
source venv/bin/activate
pip install qrcode[pil]==7.4.2
deactivate
```

### 6. 重启服务

```bash
sudo systemctl restart background_remover_tool
```

## ✅ 功能特点

1. **真实微信登录**：使用微信开放平台的OAuth2.0授权
2. **自动注册**：新用户扫码后自动创建账号
3. **自动登录**：已注册用户扫码后自动登录
4. **二维码显示**：前端自动生成并显示二维码
5. **实时轮询**：自动检测用户扫码状态
6. **用户信息**：自动获取微信昵称和头像

## 🔄 登录流程

1. 用户点击"微信登录"
2. 前端请求生成二维码
3. 后端生成微信授权URL并转换为二维码
4. 前端显示二维码
5. 用户用微信扫描二维码
6. 用户在微信中确认授权
7. 微信回调服务器，返回授权码
8. 后端用授权码换取用户信息
9. 前端轮询检查登录状态
10. 登录成功，自动刷新页面

## ⚠️ 注意事项

1. **回调域名必须配置**：否则微信无法回调您的服务器
2. **HTTPS要求**：生产环境建议使用HTTPS
3. **AppSecret保密**：不要将AppSecret提交到Git仓库
4. **测试环境**：可以使用微信开放平台的测试账号进行测试

## 🐛 常见问题

### Q: 提示"微信登录未配置"
A: 检查 `.env` 文件中是否设置了 `WECHAT_APPID` 和 `WECHAT_APPSECRET`

### Q: 扫码后提示"redirect_uri参数错误"
A: 检查微信开放平台中配置的回调域名是否正确

### Q: 二维码显示失败
A: 检查是否安装了 `qrcode[pil]` 库

### Q: 登录后没有获取到用户信息
A: 检查微信开放平台中应用的权限设置，确保已开启"获取用户信息"权限

### Q: 申请 AppID 需要收费吗？
A: 是的，需要先完成开发者资质认证，认证费用为 **300元人民币**（一次性费用）。认证通过后才能创建应用并获取 AppID。仅用于微信登录功能时，AppID 本身不额外收费。
