## 🔧 技术实现

### 后端（Flask）

**文件**：`sk_app.py`

**API路由**：
```python
@app.route('/api/tools/keyword-analyzer', methods=['POST'])
def keyword_analyzer():
    # 处理关键词分析请求
    # 支持action: extract, competition, trend, compare, longtail
```

**模拟数据函数**：
- `generate_mock_keyword_extract()` - 关键词提取
- `generate_mock_competition_data()` - 竞争度查询
- `generate_mock_trend_data()` - 趋势分析
- `generate_mock_comparison_data()` - 竞品对比
- `generate_mock_longtail_keywords()` - 长尾关键词挖掘

**使用限制**：
- 已在`PLAN_DAILY_LIMITS`中配置`keyword_analyzer`的使用次数限制
- 免费版：10次/天
- 基础版：50次/天
- 专业版：500次/天
- 旗舰版/企业版：无限制

---

### 前端（JavaScript）

**文件**：`frontend/js/app.js`

**主要函数**：
- `processKeywordAnalysis()` - 处理关键词分析请求
- `showKeywordAnalysisResult()` - 显示分析结果
- `generateKeywordExtractHTML()` - 生成关键词提取结果HTML
- `generateCompetitionHTML()` - 生成竞争度查询结果HTML
- `generateTrendHTML()` - 生成趋势分析结果HTML
- `generateCompareHTML()` - 生成竞品对比结果HTML
- `generateLongtailHTML()` - 生成长尾关键词挖掘结果HTML
- `exportKeywords()` - 导出关键词数据
- `exportToCSV()` - CSV导出函数

**界面选项**：
- 功能选择下拉框（5个功能）
- 产品描述输入框
- 平台选择下拉框（Amazon, eBay, Temu, Shopee, 所有平台）
- 竞品ASIN输入框（仅竞品对比功能显示）

## 📋 使用说明

### 1. 启动服务器

**前端**（端口8000）：
```bash
cd frontend
python -m http.server 8000
```

**后端**（端口5000）：
```bash
python sk_app.py
```

### 2. 访问工具

1. 打开浏览器访问：http://localhost:8000
2. 登录账号
3. 点击"关键词分析"工具卡片
4. 选择功能（关键词提取、竞争度查询等）
5. 输入产品描述或关键词
6. 选择目标平台（可选）
7. 点击"开始分析"
8. 查看分析结果
9. 点击"导出Excel"或"导出CSV"下载数据

## 🔑 后续接入真实API

### 当前状态

✅ **使用模拟数据**，所有功能都已完整实现，可以正常测试和使用。

### 接入真实API

**需要申请API密钥**：

1. **OpenAI API密钥**（用于GPT关键词提取和长尾关键词挖掘）
   - 申请地址：https://platform.openai.com/
   - 配置环境变量：`OPENAI_API_KEY`

2. **亚马逊SP-API凭证**（用于竞争度查询、趋势分析、竞品对比）
   - 申请地址：https://developer.amazon.com/sp-api/
   - 配置环境变量：
     - `AMAZON_SP_API_CLIENT_ID`
     - `AMAZON_SP_API_CLIENT_SECRET`
     - `AMAZON_SP_REFRESH_TOKEN`
     - `AMAZON_MARKETPLACE_ID`

**接入步骤**：

1. 申请API密钥（参考`关键词分析模块实现说明.md`）
2. 在`.env`文件中配置API密钥
3. 修改`sk_app.py`中的模拟函数，接入真实API
4. 测试真实数据功能

**代码位置**：
- 模拟函数在`sk_app.py`中（函数名以`generate_mock_`开头）
- 可以直接替换这些函数的内容，接入真实API调用
